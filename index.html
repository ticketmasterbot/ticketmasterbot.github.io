About TicketMaster Bot

TicketMaster is a Discord bot that allows users to create and manage support tickets within their server. With TicketMaster, you can easily handle user requests and issues in an organized and efficient manner. Whether you run a gaming community or a business, TicketMaster is the perfect solution for all your support needs.

Commands

Below is a list of commands that you can use with TicketMaster:

Command Description
/create ticket Create a Ticket
/close ticket Close an existing support ticket
/ticket embed Creates a ticket embed so users can click a button to create a ticket
/delete ticket Deletes the current ticket
